============
Byte Bot API
============

The Byte Bot API provides a Discord bot, alongside a web application for monitoring the status of the bot
and its servers.

.. seealso:: It is built on the `Litestar <https://litestar.dev>`_ framework
    and `discord.py <https://discordpy.readthedocs.io/en/latest/>`_.
